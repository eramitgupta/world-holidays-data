# Contributing Guide

Thank you for considering contributing to **World Holidays Data**! 🎉

---

## 🛠 How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/eramitgupta/world-holidays-data.git
cd world-holidays-data
```

### 2. Install Dependencies

```bash
pip install holidays pycountry
```

### 3. Make Changes

- Edit `index.py` to improve data generation logic
- Update `README.md` for documentation improvements
- Add new years or fix incorrect holiday entries

### 4. Run & Verify

```bash
python3 index.py
```

### 5. Submit a Pull Request

- Describe what you changed and why
- Keep PRs focused — one change per PR

---

## 📋 Guidelines

- Follow existing code style
- Test your changes before submitting
- Update `CHANGELOG.md` with a summary of your changes
- Be respectful and constructive in all discussions

---

## 🐛 Reporting Bugs

Open an issue with:
- A clear title and description
- Steps to reproduce
- Expected vs actual output

---

## 💡 Feature Requests

Open an issue with the label `enhancement` and describe what you'd like to see.

Thank you for helping make this project better! 🌍
