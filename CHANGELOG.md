## [1.0.0] - 2026-07-03

### Added
- Initial release of World Holidays Data
- Holiday data for **493 countries** from **2015 to 2059**
- Folder structure: `holidays/{country}/{year}/holidays.json`
- `countries.json` with country codes and full names
- `index.py` — single script to generate all data
- README, LICENSE, and contributing guidelines
