# Support

Thank you for using **World Holidays Data**! Here's how to get help.

---

## 📖 Documentation

Start with the [README](README.md) for setup and usage instructions.

---

## 🐛 Found a Bug?

Open an issue on GitHub with:
- A clear title
- Steps to reproduce the problem
- Expected vs actual behavior
- Your Python version (`python3 --version`)

---

## 💡 Feature Request?

Open an issue with the label `enhancement` and describe what you'd like added.

---

## ❓ General Questions

For general questions, open a [GitHub Discussion](https://github.com/eramitgupta/world-holidays-data/discussions).

---

## ⚠️ Known Limitations

- **India (IN / IND):** Holidays available only from 2001–2035 (library limitation)
- **Marshall Islands (MH / MHL):** Holidays available only from 2020 onwards
- Some country codes appear in both alpha-2 and alpha-3 formats

---

We'll do our best to respond within **2–3 business days**. 🙏
